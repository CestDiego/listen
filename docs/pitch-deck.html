The Problem

Voice assistants are blind between commands

  • They respond to explicit commands but miss implicit signals
  • Users experiencing distress, burnout, or mood shifts get zero support
  • No continuous understanding of user state across time
  • Reactive by design — they wait for you to ask

The Solution

Continuous observability, not commands

  • Continuous mic transcription → real-time intent classification
  • 6-dimensional user state vector tracking mood, energy, engagement, focus, music intent, and wellbeing
  • Proactive skill activation with hysteresis-based safety gates
  • All running locally on Apple Silicon — privacy-first, zero cloud dependencies

Safety Layer

The Activation Gate

In vigilant state: promotes target skill at 0.40 confidence

Cost-asymmetric: missing genuine distress costs 10× a false positive check-in
